    Project Idea

    Overview
    My project idea is a Covid-19 Check-in app, where users will be able to scan QR/enter access codes to check in a particular place by using the mobile’s location, time, contact and name. Businesses will need to register their name with the tracking app in order to generate a QR/access code for users to check in, and it will be as simple as entering a form that will be able to save the users’ details for further use. The app will be able to track when, who and where signed into a place in order to help track-down any potential covid-19 cases.
    Motivation
    The idea of this project came from when restaurants/businesses started re-opening after lockdown and used pen and paper to track individuals. I thought it would be a great idea to create an application that gives the user’s time, location and contact details at the click of a button or scanning of a QR code to aid with contact tracing and to keep individuals safe. Not long ago, Service Victoria created a check-in app that is very similar to my project idea.
    Description
  • The application will be free on Android (Google Play) & IOS (App store), provided users have a wifi/internet connection to download the app.
  • Users must be over 12 years of age to register, and any children will be required to check-in via their guardian’s check-in process as there will be a prompt asking if there are any dependent child(s) with them.
  • The application will be modern and easy to navigate, with a simple ‘fill-the-survey’ type navigation and it’s done. The application will prompt everything the user has to input, and users/businesses will be registered easily.
  • Users will need to grant the app access to its location, whether it be while the app is open or unlimited access.
  • Users will need to enter the name, contact details and address at the time of registration - but when scanning/inputting the code the details will automatically be filled in and they will only need to press ‘Check-in’.
  • Mobile phones will need to have a QR scanning function with their camera in order to scan, however an access code will also be provided in case users do not have this function.
  • A QR Code/Access code will be provided by businesses that users can either scan or enter
  • Businesses will need to register their location(s), and primary contact (whether it be the manager, owner), and the app will generate a QR code that each location will be able to use indefinitely.
  • After businesses register, the QR/access code poster will be sent to the user's email and they can then print out the code to display in the front of their building.
  • Dark/light mode will be added to help visibility, and aid in appearances.
  • Only your name and phone number will be provided at each check-in process to help secure privacy, but registering an account will require personal details.
  • To secure users information and privacy, if there are no cases/contact tracing linked within 14 days of the building the users check-in details will automatically be deleted.
